Telegram Channel Automation (Selenium)

This project is a basic Telegram Web automation script built using Python + Selenium.
It opens Telegram Web using a saved session, navigates to a specific channel, scrolls messages, and sends random human-like messages.

Features

Uses saved Telegram session (no OTP every run)

Opens Telegram Web automatically

Navigates directly to a target channel

Scrolls up to older messages (human-like behavior)

Sends random messages from a predefined list

Includes delays to reduce bot-like activity

Tech Stack

Python

Selenium

Chrome WebDriver
