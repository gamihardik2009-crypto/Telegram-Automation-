Telegram Channel Automation (Selenium)
Overview
This project is a Telegram Web automation script built using Python and Selenium. It opens Telegram Web using a saved session, navigates to a specific channel, scrolls through messages, and sends random human-like messages.

Features
✅ Uses saved Telegram session (no OTP every run)

✅ Opens Telegram Web automatically

✅ Navigates directly to a target channel

✅ Scrolls up to older messages (human-like behavior)

✅ Sends random messages from a predefined list

✅ Includes delays to reduce bot-like activity

Tech Stack
Python

Selenium

Chrome WebDriver

Installation & Setup
Prerequisites
Python 3.7+

Chrome browser installed

ChromeDriver matching your Chrome version
